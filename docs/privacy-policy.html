Privacy Policy – Greenhouse Security Code Helper

Last updated: [DATE]

Greenhouse Security Code Helper (“the Extension”) is a Chrome browser extension that helps you copy security codes from Greenhouse application emails in your Gmail inbox.

Data we use

  • Gmail: The Extension requests read-only access to your Gmail via Google’s Gmail API. It only looks for emails from Greenhouse that contain a security code and reads the code from the email body. It does not read, store, or send any other email content.
  • Stored on your device: The Extension stores your preferences (e.g. notification settings, check interval) and the most recently extracted security code and timestamp in your browser’s local storage. This data never leaves your device to our servers.
  • Google sign-in: Sign-in uses Chrome’s identity API (Google OAuth). We do not handle or store your password; Chrome and Google manage authentication.

What we do not do

  • We do not sell or share your data with third parties.
  • We do not send your emails or security codes to any server we control. Processing happens on your device; only Google’s Gmail API is called to fetch your emails.
  • We do not use your data for advertising, credit, or any purpose unrelated to the Extension’s single purpose (showing and copying Greenhouse security codes).

Host permissions

The Extension may request access to all websites (“Host permission”) only so it can show an optional in-page notification (toast) on whatever tab you have open when a new code is detected. It does not collect or read content from those sites.

Your choices

You can revoke Gmail access at any time in your Google Account permissions. Removing the Extension from Chrome stops all access and deletes locally stored data.
